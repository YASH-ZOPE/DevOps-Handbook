# Linux File System

## Filesystem Hierarchy

In Linux, everything starts with the **Root directory (`/`)**.

```text
/ (Root)
│
├── home
├── bin
├── etc
├── usr
├── var
└── root
```

---

## Standard Directories Overview

| Directory | Description |
|-----------|-------------|
| `/` | **Root**: Top-level directory of the entire filesystem hierarchy. |
| `/home` | User home directories for personal files and settings. |
| `/bin` | Essential command binaries (e.g., `ls`, `cp`, `cat`). |
| `/etc` | System configuration files. |
| `/usr` | User programs and secondary system utilities. |
| `/var` | Variable files (log files, cache, databases). |
| `/root` | Home directory for the System Administrator (**root** user). |
