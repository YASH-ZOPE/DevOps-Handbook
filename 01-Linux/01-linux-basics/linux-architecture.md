# Linux Architecture

## Kernel

The **Kernel** is the core component of the Operating System.

- It manages all hardware resources (CPU, Memory, I/O devices).
- It controls process scheduling and system memory allocation.

---

## Shell

The **Shell** is an interface used to communicate with the Kernel.

- It acts as a gateway between the user and the Kernel.
- It translates user commands into instructions the Kernel can execute.

---

## Bootloader & GRUB

### Bootloader

The **Bootloader** loads all required OS files into memory during system startup.

### GRUB

- **GRUB** stands for **GRand Unified Bootloader**.
- It is the standard bootloader used in Linux distributions.
- It initializes and manages the Linux boot process.
