# Linux Basics

## What is Linux?

Linux is an open-source operating system created by **Linus Torvalds**.

Most servers and applications use Linux because it is:

- **Secure**
- **Stable**
- **Free**

Linux is mainly used in:

- Cloud Computing
- DevOps
- Server Infrastructure

---

## UNIX vs Linux

| Feature         | UNIX               | Linux                           |
|-----------------|--------------------|---------------------------------|
| **Cost**        | Paid / Proprietary | Free / Open-source              |
| **Source Code** | Closed-source      | Open-source                     |
| **Licensing**   | Commercial License | GNU General Public License (GPL)|

---

## Windows vs Linux

| Feature              | Windows                          | Linux                                            |
|----------------------|----------------------------------|--------------------------------------------------|
| **Licensing**         | Commercial License               | GNU General Public License (GPL)                 |
| **Primary Use Case** | General-purpose OS               | Servers, networking, dev environments, terminals |
| **Security**         | Usually requires antivirus       | Generally no antivirus required                  |
| **Updates**          | Scheduled OS updates             | Independent package & kernel updates             |
| **Remote Access**    | Uses RDP (Remote Desktop)        | Uses SSH (Secure Shell)                          |

## Application Architectures

### Standalone Application

A standalone application:

- Runs locally on a single computer.
- Does not require a Web Server.
- Does not require an Application Server.

**Examples:**
- Calculator
- Notepad
- VLC Media Player

---

### Web Application

A web application:

- Runs inside a web browser over a network connection.
- Uses multiple backend components such as:
  - **Web Server**
  - **Application Server**
  - **Database**

**Examples:**
- Gmail
- Amazon
