# Server and Client

## Server & Client

### Server

A **server** is a computer that serves data or services to a client.

### Client

A **client** is a device or software that requests data from a server.

---

## Communication Flow

```text
Client
  │
Request
  │
  ▼
Server
  │
Response
  │
  ▼
Client
```

---

## Web Server

A **web server** serves **static content**.

### Static Content

- HTML
- CSS
- JavaScript
- Images
- Videos

### Example

- **Nginx**

---

## Application Server

An **application server** serves **dynamic content**. It processes user requests and generates dynamic responses.

### Dynamic Content

- Login / Authentication
- User Profile
- Dashboard Data

### Example

- **Node.js**

---

## How the Internet Works

```text
Data Center
     │
     ▼
Internet Backbone
     │
     ▼
Undersea Fiber Optic Cable
     │
     ▼
ISP (Internet Service Provider)
     │
     ▼
Router
     │
     ▼
Laptop / Mobile Device
```

### Explanation

- **Data Centers**: Applications are hosted inside Data Centers.
- **Undersea Fiber Optic Cables**: Data centers across countries are connected through undersea fiber optic cables.
- **ISP**: Internet Service Providers (ISPs) provide internet connectivity to users.
- **Router**: Routers distribute the internet connection to user devices.
